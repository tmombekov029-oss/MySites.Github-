О проекте Oracle Audit

Этот учебный веб-проект посвящён теме аудита действий пользователей в Oracle Database. Он создан как демонстрационная информационная платформа, содержащая теорию, SQL-примеры, схемы, визуальные материалы и практические рекомендации по анализу аудита.

Цели проекта

  • Предоставить понятное объяснение механизмов аудита в Oracle Database
  • Показать разницу между Unified, Classic и Fine-Grained Auditing
  • Обучить работе с инструментами анализа логов аудита
  • Дать реальные примеры SQL-запросов для расследования действий пользователей
  • Показать современные практики безопасности Oracle

Структура сайта

  • Главная — краткое описание темы аудита
  • Unified Auditing — современный механизм аудита в Oracle
  • FGA — аудит на уровне строк и колонок
  • Classic Auditing — традиционный способ аудита
  • Инструменты — SQL Developer, OEM, SIEM, CLI
  • Примеры SQL — готовые запросы для анализа логов
  • Рекомендации — лучшие практики ИБ
  • О проекте — данная страница

Использованные технологии

  • HTML5 — структура страниц
  • CSS3 — визуальное оформление, адаптивная верстка
  • JavaScript — работа бургер-меню, анимации
  • SVG-схемы — для отображения архитектуры Oracle
  • Oracle SQL — для демонстрации реальных запросов
